MÓDULO 1: SISTEMAS

Concepto de Sistema

ENTRADA

Datos que se capturan o reciben del entorno

PROCESO

Transformación de datos a información

SALIDA

Información útil para los usuarios

"Un sistema es un conjunto de elementos interrelacionados para lograr un mismo objetivo."Bertalanffy, L. von (1945)

Análisis y Diseño

📊 Análisis de Sistemas

Identificar QUÉ debe hacer el sistema.

  • Identificar problemas existentes
  • Recopilar datos y requisitos
  • Evaluar soluciones actuales
  • Documentar el sistema actual

✏️ Diseño de Sistemas

Especificar CÓMO hará el sistema lo que se necesita.

  • Definir arquitectura del sistema
  • Diseñar módulos y componentes
  • Crear prototipos
  • Planificar la implementación

¿Quiénes son los Usuarios?

👨‍💼 Usuarios Finales

Usan el sistema diariamente en sus tareas.

👥 Usuarios Gerenciales

Toman decisiones basadas en la información.

💻 Técnicos

Desarrollan, programan y mantienen el sistema.

🏢 Stakeholders

Clientes, proveedores, socios e inversores.

Elementos y Características

🔧 Características de los Sistemas

  • Holismo: El todo es mayor que la suma de partes
  • Equifinalidad: Diversos caminos para un mismo fin
  • Homeostasis: Capacidad de autorregulación
  • Adaptación: Se ajustan al entorno
  • Entropía: Tendencia al desorden

📦 Tipos de Sistemas

  • Naturales: Sistema solar, ecosistema
  • Artificiales: Sistema computarizado
  • Abiertos: Empresa, organismo vivo
  • Cerrados: Reloj mecánico
  • Conceptuales: Teoría matemática

MÓDULO 2: INFORMACIÓN

Definición de Información

📖 Teoría de la Información (Shannon, 1948)

"La información es la medida de la reducción de la incertidumbre que proporciona un mensaje."Shannon, C. E. (1948)

🏢 En Ingeniería de Sistemas

"La información son datos procesados y organizados que tienen significado y utilidad para la toma de decisiones."

Características de la Información

⏱️ Oportunidad

Disponible cuando se necesita.

🎯 Precisión

Correcta y confiable.

📝 Relevancia

Pertinente al propósito.

🔍 Comprensibilidad

Clara y fácil de interpretar.

📦 Completitud

Incluye todos los datos necesarios.

🔄 Accesibilidad

Obtenida fácilmente.

MÓDULO 3: SISTEMAS DE INFORMACIÓN

Definición

"Un sistema de información es el conjunto de agentes, códigos y procesos que interactúan coordinadamente entre sí con un fin común o propósito."

🖥️ Hardware

Equipo físico para procesar y almacenar datos.

💿 Software

Programas y procedimientos de procesamiento.

📊 Datos

Información que representa las actividades.

🌐 Red

Infraestructura de comunicación.

👥 Personas

Desarrollan, mantienen y usan el sistema.

📋 Procesos

Procedimientos y reglas de operación.

Operaciones con un SI

📥 Captura
Registrar datos
🔄 Procesamiento
Transformar
💾 Almacenamiento
Guardar
🔍 Recuperación
Consultar
📤 Salida
Reportes

🔄 Operaciones Principales

  • Input: Capturar datos crudos
  • Proceso: Calcular, clasificar, ordenar
  • Almacenamiento: Guardar en BD o archivos
  • Output: Generar reportes, pantallas
  • Feedback: Evaluar y corregir

📋 Ciclo de Vida del SI

  1. Conocer la organización
  2. Identificar problemas y oportunidades
  3. Diagnóstico
  4. Determinar necesidades
  5. Propuesta
  6. Diseño del sistema
  7. Codificación
  8. Implementación
  9. Mantenimiento

Tipos de Sistemas de Información

Modelo de la Pirámide Organizacional

🏢 EIS - Información Ejecutiva
📊 DSS - Soporte a Decisiones
📈 MIS - Información Gerencial
📋 TPS - Procesamiento de Transacciones
⚙️ OAS - Automatización de Oficinas

Desde operaciones diarias (base) hasta decisiones estratégicas (cima)

Detalle de los Tipos de SI

📋 TPS

Registra operaciones diarias: ventas, inventario, nómina.

📈 MIS

Genera reportes periódicos para gerentes.

📊 DSS

Análisis de escenarios y simulaciones.

🏢 EIS

Información de alto nivel para C-suite.

Otros Tipos de Sistemas de Información

🤖 Sistema Experto

Emula experto humano con IA y reglas de inferencia.

📦 ERP

Integra toda la información y procesos de la organización.

🌐 SIG

Análisis de información georreferenciada.

📝 OAS

Optimizan el trabajo administrativo.

🏢 SI Estratégico

TI para respaldar estrategia competitiva.

☁️ SI en la Nube

Trabajo remoto y escalabilidad.

Información y Toma de Decisiones

📊 Niveles de Decisión

🏢 Estratégica: Largo plazo, Alta Dirección
📋 Táctica: Mediano plazo, Gerencia Media
⚙️ Operativa: Corto plazo, Supervisores

💡 Calidad de la Información

  • Relevante: Perteneciente al problema
  • Oportuna: Disponible a tiempo
  • Exacta: Sin errores
  • Completa: Sin omisiones
  • Económica: Costo-beneficio favorable
