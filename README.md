## Descripción del Proyecto

Este proyecto es una aplicación web interactiva que cubre los siguientes módulos temáticos:

### Módulo 1: Sistemas
- Concepto de sistema
- Análisis y diseño de sistemas
- Usuarios de un sistema
- Elementos, componentes y características
- Tipos de sistemas

### Módulo 2: Información
- Definición de información
- Características de la información
- Tipos de información
- Proceso de tratamiento de datos

### Módulo 3: Sistemas de Información
- Definición de sistema de información
- Operaciones con un sistema de información
- Tipos de sistemas de información
- La información y la toma de decisiones

## Componentes

### 1. Juego del Laberinto (`game.html`)
- Laberinto generado algoritmicamente con paredes visibles
- 15 conceptos coleccionables distribuidos por módulo
- Sistema de preguntas al encontrar obstáculos
- Enemigos que patrullan el laberinto
- Llaves, puertas bloqueadas y minas
- Niveles de dificultad progresiva
- Minimapa en tiempo real
- Controles: WASD / Flechas / Touch

### 2. Presentación (`presentacion.html`)
- 17 diapositivas con animaciones SVG
- Diagramas de flujo animados (Sistema, Información, Datos)
- Pirámide organizacional interactiva
- Secciones expandibles
- Navegación por teclado (← →) y botones
- Transiciones suaves entre slides

### 3. Quiz Interactivo (`quiz.html`)
- 10 preguntas de selección múltiple
- Retroalimentación inmediata (correcto/incorrecto)
- Puntuación final al completar
- Navegación entre secciones

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos, animaciones, transiciones, grid, flexbox
- **JavaScript** - Lógica del juego, navegación, interactividad
- **Canvas API** - Renderizado del laberinto y minimapa
- **SVG** - Diagramas animados en la presentación

---

## Cómo Ejecutar

1. Descargar o clonar el repositorio
2. Abrir `index.html` en un navegador web moderno
3. Seleccionar una de las tres secciones:
   - **Juego del Laberinto** - Explorar conceptos jugando
   - **Presentación** - Recorrer las diapositivas
   - **Quiz** - Evaluar conocimientos

> **Nota:** No requiere servidor local. Funciona directamente abriendo los archivos HTML.
