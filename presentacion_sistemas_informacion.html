MÓDULO 1: SISTEMAS

Concepto de Sistema

"Un sistema es un conjunto de elementos interrelacionados para lograr un mismo objetivo."— Ludwig von Bertalanffy (1945)

📌 Ejemplo

Una universidad: estudiantes, profesores, administrativos trabajan coordinadamente para formar profesionales.

Análisis y Diseño

📊 Análisis

Estudiar procedimientos para identificar QUÉ debe hacer el sistema.

  • Identificar problemas
  • Recopilar requisitos
  • Evaluar soluciones
  • Documentar sistema actual

💡 Autores

Sommerville: "Análisis = comprensión"
Pressman: "¿QUÉ debe hacer?"

✏️ Diseño

Especificar arquitectura, componentes, módulos.

  • Arquitectura del sistema
  • Módulos e interfaces
  • Prototipos
  • Plan de implementación

💡 Diferencia

Análisis: QUÉ  |  Diseño: CÓMO

¿Quiénes son los Usuarios?

👨‍💼

Finales

Usan el sistema diariamente.

👥

Gerenciales

Toman decisiones con la info.

💻

Técnicos

Desarrollan y mantienen.

🏢

Stakeholders

Clientes, proveedores, socios.

Elementos y Componentes

🔧 Características

  • Holismo: Todo > suma partes
  • Equifinalidad: Múltiples caminos
  • Homeostasis: Autorregulación
  • Adaptación: Se ajusta al entorno
  • Entropía: Tendencia al desorden

MÓDULO 2: INFORMACIÓN

Definición

📖 Shannon (1948)

"Medida de la reducción de incertidumbre que proporciona un mensaje."

🏢 Ingeniería de Sistemas

"Datos procesados y organizados con significado y utilidad."

Dato → Información → Conocimiento

Características

⏱️ Oportunidad

Disponible cuando se necesita.

🎯 Precisión

Correcta y confiable.

📝 Relevancia

Pertinente al propósito.

🔍 Comprensibilidad

Clara y fácil.

📦 Completitud

Todos los datos necesarios.

🔄 Accesibilidad

Obtenida fácilmente.

Información y Decisiones

La información es el insumo fundamental para la toma de decisiones.

📊 Niveles

🏢 Estratégica▼

Largo plazo: expansión, nuevos mercados.

📋 Táctica▼

Mediano plazo: recursos, presupuestos.

⚙️ Operativa▼

Corto plazo: horarios, inventario.

💡 Calidad

  • Relevante
  • Oportuna
  • Exacta
  • Completa
  • Económica

⚠️ Paradigma

"Demasiada info puede ser tan dañina como muy poca."

💎 Información

Datos con significado. Su calidad determina decisiones.
